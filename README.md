# Currency Converter

A small MVP web app that converts currencies using live exchange-rate data.

## Features

- amount input
- source and target currency selectors
- swap action
- live exchange rates fetched from a public API
- loading and error states
- responsive UI

## Tech stack

- React + TypeScript
- Vite
- Vitest + Testing Library
- GitHub Actions for PR CI

## Getting started

```bash
cd frontend
npm install
npm run dev
```

## Testing

```bash
cd frontend
npm test
```

## Build

```bash
cd frontend
npm run build
```
